---
name: ada-exception-check
description: Check whether a specific resource is eligible for an ADA Title II compliance exception before investing in remediation.
argument-hint: <path>
---

Run exceptions-analyst on the resource at the path provided. Evaluate all five exception categories: archived content, preexisting documents, password-protected, preexisting social media post, third-party content. Report which criteria are met or not met and flag any items requiring human verification.
