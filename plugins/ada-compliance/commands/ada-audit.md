---
name: ada-audit
description: Run a full ADA Title II / WCAG 2.1 AA audit on a path. Dispatches compliance-lead to scope, classify, and audit all content types.
argument-hint: [path]
---

Use compliance-lead to run a full ADA Title II / WCAG 2.1 AA audit on the path provided by the user (default: repo root if no path given). compliance-lead will scope the audit, classify files by type, dispatch specialist agents, run exceptions-analyst across all resources, and produce a consolidated report with compliance matrix.
