# ADA Compliance Plugin

Review content and get guidance on WCAG 2.1 AA / ADA Title II compliance with specialist reviewer agents for web content, visual design, documents, and audio/video.

## Versioning

This plugin targets WCAG 2.1 AA (per ADA Title II as of 2026). A future skill `wcag-2.2-migration` will map WCAG 2.1 → 2.2 additions when the underlying regulation moves.
